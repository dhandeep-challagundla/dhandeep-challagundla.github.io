# Dhandeep Challagundla — GitHub Pages Website

## Publish the website for free

### 1. Create a GitHub repository
Create a new **public** repository. A simple name is:

`digital-business-card`

### 2. Upload the site files
Upload the contents of this folder to the repository root:

- `index.html`
- `dhandeep-challagundla.vcf`
- the complete `assets` folder

Do not upload only the ZIP file. Extract it first, then upload the files and folder.

### 3. Turn on GitHub Pages
In the repository:

1. Open **Settings**
2. Select **Pages**
3. Under **Build and deployment**, choose **Deploy from a branch**
4. Select the `main` branch and `/ (root)`
5. Click **Save**

GitHub will provide an address similar to:

`https://YOUR-USERNAME.github.io/digital-business-card/`

It may take a few minutes to appear.

### 4. Test before printing
Open the site on a phone and test:

- Save Contact
- Email
- Phone
- LinkedIn
- Rezonent website

Use the final GitHub Pages URL to generate the QR code. Keep the URL unchanged after printing the business cards.

## Updating the page
Edit or replace files in the repository. The GitHub Pages URL stays the same, so the printed QR code continues to work.
